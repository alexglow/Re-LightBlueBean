# Re-LightBlueBean
Reviving [PunchThrough Design](https://punchthrough.com)'s [LightBlue Bean](https://punchthrough.com/bean), after the shutdown. Because this beautiful little Bluetooth board deserves to live :D  

I've written up a tutorial for Mac users on my [website page](https://alexglow.notion.site/Beanie-Bots-19a5bc4fe798809aa190db6e0d67d61d?source=copy_link), which will likely update far more often than this repo. As of May '26, feel free to hit me up on [Mastodon](https://chaos.social/@alexglow) or [Bluesky](https://bsky.app/profile/alexglow.bsky.social) for updates.

I'm especially interested in backing up the Bean+ files, if anyone has them – since I was mostly interested in the original Bean.

## Hardware files
When PunchThrough discontinued the LightBlue Bean and Bean+, they open-sourced the files. What you see here is the ones I grabbed for the former, as well as a file with some of their notes.

## Code examples
Unless otherwise noted, examples were written by the PunchThrough team and are in the public domain. This info should be in the files, but I'm trying to get some basic material up immediately, and then I'll double-check.

I'm planning to upload two versions: a backup of their originals, and new versions I've been able to compile and upload in 2026 (using the PlatformIO plugin in VSCode).
