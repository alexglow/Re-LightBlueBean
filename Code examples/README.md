# Code examples for LightBlue Bean

### Tested LightBlue Bean demos
Some examples I've cobbled together from other sources – including my own old Hackster project(s) and example Arduino code.

### Original PunchThrough demos
Pulled from the original PunchThrough site (via the Internet Archive). I added the public-domain header, along with the exact URL each was pulled from. These will probably work, using [this updated process](https://alexglow.notion.site/Beanie-Bots-19a5bc4fe798809aa190db6e0d67d61d?source=copy_link), as long as you add **#include <Arduino.h>** at the top.
